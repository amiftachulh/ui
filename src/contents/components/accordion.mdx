## Installation

<Steps>

<Step>Install the following dependencies:</Step>

<PackageInstaller name="@radix-ui/react-accordion" />

<Step>Add keyframes to your preset then add it to semantic tokens.</Step>

```ts highlight="3-10"
export const keyframes = defineKeyframes({
  // ... other keyframes
  "accordion-down": {
    from: { height: 0 },
    to: { height: "var(--radix-accordion-content-height)" },
  },
  "accordion-up": {
    from: { height: "var(--radix-accordion-content-height)" },
    to: { height: 0 },
  },
});
```

```ts highlight="4,5"
export const semanticTokens = defineSemanticTokens({
  // ... other semantic tokens
  animations: {
    "accordion-down": { value: "accordion-down {durations.normal} {easings.out}" },
    "accordion-up": { value: "accordion-up {durations.normal} {easings.out}" },
  },
});
```

<Step>Copy and paste the following code into your project.</Step>

## Usage

```tsx
import {
  Accordion,
  AccordionContent,
  AccordionItem,
  AccordionTrigger,
} from "@/components/ui/accordion";
```

```tsx
<Accordion type="single" collapsible>
  <AccordionItem value="item-1">
    <AccordionTrigger>Is it accessible?</AccordionTrigger>
    <AccordionContent>Yes. It adheres to the WAI-ARIA design pattern.</AccordionContent>
  </AccordionItem>
</Accordion>
```
