## Installation

<Steps>

<Step>Install the following dependencies:</Step>

<PackageInstaller name="@radix-ui/react-alert-dialog" />

<Step>Add the [Button](/docs/components/toggle) component to your project.</Step>

The `AlertDialog` component uses the `button` recipe. Make sure you have it installed in your project.

<Step>Copy and paste the following code into your project.</Step>

<ComponentSource name="alert-dialog.ts" type="slot-recipe" />

<ComponentSource name="alert-dialog.tsx" type="component" />

<Step>Import the recipe into the respective `index.ts` file.</Step>

```tsx
import { alertDialogSlotRecipe } from "./alert-dialog";

export const slotRecipes = {
  alertDialog: alertDialogSlotRecipe,
};
```

<Step>Run `panda codegen` and update the import paths to match your project setup.</Step>

</Steps>

## Usage

```tsx
import {
  AlertDialog,
  AlertDialogAction,
  AlertDialogCancel,
  AlertDialogContent,
  AlertDialogDescription,
  AlertDialogFooter,
  AlertDialogHeader,
  AlertDialogTitle,
  AlertDialogTrigger,
} from "@/components/ui/alert-dialog";
```

```tsx
<AlertDialog>
  <AlertDialogTrigger>Open</AlertDialogTrigger>
  <AlertDialogContent>
    <AlertDialogHeader>
      <AlertDialogTitle>Are you absolutely sure?</AlertDialogTitle>
      <AlertDialogDescription>
        This action cannot be undone. This will permanently delete your account and remove your data
        from our servers.
      </AlertDialogDescription>
    </AlertDialogHeader>
    <AlertDialogFooter>
      <AlertDialogCancel>Cancel</AlertDialogCancel>
      <AlertDialogAction>Continue</AlertDialogAction>
    </AlertDialogFooter>
  </AlertDialogContent>
</AlertDialog>
```
