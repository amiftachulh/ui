## Installation

<Steps>

<Step>Install the following dependencies:</Step>

<PackageInstaller name="@radix-ui/react-avatar" />

<Step>Copy and paste the following code into your project.</Step>

<ComponentSource name="avatar.ts" type="slot-recipe" />

<ComponentSource name="avatar.tsx" type="component" />

<Step>Import the recipe into the respective `index.ts` file.</Step>

```tsx
import { avatarSlotRecipe } from "./avatar";

export const slotRecipes = {
  avatar: avatarSlotRecipe,
};
```

<Step>Run `panda codegen` and update the import paths to match your project setup.</Step>

</Steps>

## Usage

```tsx
import { Avatar, AvatarFallback, AvatarImage } from "@/components/ui/avatar";
```

```tsx
<Avatar>
  <AvatarImage src="https://github.com/amiftachulh.png" />
  <AvatarFallback>A</Avatar.Fallback>
</Avatar>
```
