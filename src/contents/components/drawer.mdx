## Usage

```tsx
import { Drawer } from "@/components/ui/drawer";
```

```tsx
<Drawer>
  <DrawerTrigger>Open</DrawerTrigger>
  <DrawerContent>
    <DrawerHeader>
      <DrawerTitle>Are you absolutely sure?</DrawerTitle>
      <DrawerDescription>
        This action cannot be undone. This will permanently delete your account and remove your data
        from our servers.
      </DrawerDescription>
    </DrawerHeader>
  </DrawerContent>
</Drawer>
```

### Size

You can adjust the size of the drawer using CSS classes:

```tsx
<Drawer>
  <DrawerTrigger>Open</DrawerTrigger>
  <DrawerContent css={{ w: "400px", sm: { w: "540px" } }}>
    <DrawerHeader>
      <DrawerTitle>Are you absolutely sure?</DrawerTitle>
      <DrawerDescription>
        This action cannot be undone. This will permanently delete your account and remove your data
        from our servers.
      </DrawerDescription>
    </DrawerHeader>
  </DrawerContent>
</Drawer>
```
