## Installation

<Steps>

<Step>Copy and paste the following code into your project.</Step>

<ComponentSource name="textarea.ts" type="recipe" />

<ComponentSource name="textarea.tsx" type="component" />

<Step>Import the recipe into the respective `index.ts` file.</Step>

```tsx
import { textareaRecipe } from "./textarea";

export const recipes = {
  textarea: textareaRecipe,
};
```

<Step>Run `panda codegen` and update the import paths to match your project setup.</Step>

</Steps>

## Usage

```tsx
import { Textarea } from "@/components/ui/textarea";
```

```tsx
<Textarea />
```
